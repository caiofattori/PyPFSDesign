Abrir o PyPFSDesign

O PyPFSDesign utiliza o Python3 para ser executado. Para inicializá-lo pode utilizar um dos scripts de inicialização. Para sistema operacional Microsoft Windows, utiliza-se o script pfsdesing.bat que se encontra na pasta raiz do projeto. Para sistema operacional Linux, utiliza-se o script pfsdesing.sh que se encontra na pasta raiz do projeto.

Também é possível abrir a tela manualmente. Para isso, execute o comando [pasta para o python]/python3 [pasta para o projeto]/src/window.py
