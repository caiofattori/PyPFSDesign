Ajuda do PyPFSDesign

O PyPFSDesign é uma ferramenta para modelagem em Production Flow Schema (PFS).

O PFS é composto do 3 elementos básicos: atividades, distribuidores e relacionamentos. As atividades são elementos ativos, modelam eventos que alteram estados e são representadas por um texto entre colchetes. Os distribuidores são elementos passívos, modelam os estados e são representados por circunferências. Os relacionamentos são elementos de ligação, modelam o fluxo entre estados a as relações entre atividades e distribuidores, e são representados por arcos com uma seta apontada para o elemento de destino e partindo do elemento de origem.

Dos elementos básicos, alguns elementos se derivam. A partir das atividades existem a abertura de atividade e o fechamento de atividade, ambos também se comportam como elementos ativos. A partir do relacionamento existe o fluxo secundário que é um elemento utilizado para modelar envio de mensagens entre serviços complexos.

Por regra, os relacionamentos e suas derivações apenas podem ligar um elemento passivo a um elemento ativo ou um elemento ativo a um elemento passivo. Nunca é possível ligar dois elementos ativos nem dois elementos passivos.

O PFS permite também aplicar a metodologia top-down para a modelagem, uma vez que as atividades podem ser expandidas em modelos com maior grau de detalhamento.
