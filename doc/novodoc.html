Criar novo documento

O novo documento abre uma janela de edição de modelo

Para criar um, basta clicar no primeiro ícone da barra de ferramentas, na parte superior, ou no menu Arquivo->Novo ou pelo atalho Ctrl+N

Os ícones podem variar de acordo com a versão do sistema operacional

Os novos documentos não possuem arquivo associado e precisam ser salvos

para persistir.
