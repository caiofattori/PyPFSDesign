Exportar como imagem

Os gráficos do PFS são vetorizados, então é possível salvá-los como arquivos PNG ou SVG. Esse processo de salvar como uma imagem é chamado de exportar.

Para exportar um documento como imagem, basta clicar no quinto ícone da barra de ferramentas, na parte superior, ou no menu Arquivo->Exportar ou pelo atalho Ctrl+P, e selecionar o formato de destino pelo extensão do nome do arquivo escolhido (SVG ou PNG).

Os ícones podem variar de acordo com a versão do sistema operacional
