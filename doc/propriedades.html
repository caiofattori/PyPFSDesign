Tabela de propriedades

A tabela de propriedades é a tabela no lado direito superior. Quando um elemento é selecionado, suas propriedades são mostradas na tabela, sendo cada linha responsável por um dos itens da propriedade.

Para mostrar as propriedades da página, basta clicar na tela de edição fora de todos os elementos.

As propriedades de altura e de largura da página são alteradas em quatro campos sendo que podem aumentar com valores positivos ou diminuir com valores negativos. As alterações de altura para cima e largura para esquerda fazem com que os elementos presentes na página mudem as posições de elementos.

As tags são adicionadas clicando no botão + presente na linha de cada elemento.
